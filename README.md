Compared models using regression metrics and selected the best-performing model based on accuracy and generalization.

Refactored notebook experiments into modular, reusable Python code following best practices.

Deployment

Built a Flask web application to deploy the trained model and enable real-time student performance prediction through a user-friendly interface.
